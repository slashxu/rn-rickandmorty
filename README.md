## 💻 Pré-requisitos
* Expo
* async-storage
* react-navigation
* axios
* styled-components

## API
Utilizado a API de personsagens: (https://rickandmortyapi.com/)

## 🚀 Instalando Projeto

Para instalar o projeto, siga estas etapas:

Clone o repositório
```
https://github.com/slashxu/rn-rickandmorty.git
```

Baixe as dependências
```
npm install ou yarn install
```

Rodando o app
```
expo start
```
